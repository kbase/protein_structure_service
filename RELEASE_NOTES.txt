OVERVIEW
------------------------------------------
This service provides users methods for finding Protein Structure Data
Bank (PDB) entries whose protein sequences are similar to a KBase
protein sequence, specified by either a MD5 id or a feature or coding
sequence id.  Deployment documentation is in README.md in this
directory.
